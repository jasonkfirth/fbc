sfxlib example media
====================

These files are small example assets used by the sfxlib example programs.
They were downloaded from Wikimedia Commons so the examples can exercise
real file playback instead of placeholder filenames.

Local files
-----------

- `buzzer.wav`
  Source: https://commons.wikimedia.org/wiki/File:Buzzer.wav
  License: CC0
  Artist: David Pride

- `good-morning-to-all.ogg`
  Source: https://commons.wikimedia.org/wiki/File:Good_Morning_to_All.ogg
  License: Public domain
  Artist: Eirik1231

- `harmonized-scale.mid`
  Source: https://commons.wikimedia.org/wiki/File:Harmonized_scale.mid
  License: Public domain
  Artist: User:Hyacinth

- `clown-laugh.mp3`
  Source: https://commons.wikimedia.org/wiki/File:Scotcampbell_-_clown_laugh_(cc0)_(freesound).mp3
  License: CC0
  Artist: scotcampbell

If the media files are missing, run `fetch-media.ps1` from the
`examples/sfxlib` directory.
